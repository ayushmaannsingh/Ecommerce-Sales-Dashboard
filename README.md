📊 Ecommerce Sales Dashboard – Power BI Project

This project showcases an interactive Ecommerce Sales Dashboard developed in Power BI to analyze and visualize sales data effectively. The dashboard helps track important business metrics such as Sales, Profit, Quantity, and Average Order Value (AOV) while providing insights into customer behavior, payment methods, monthly profit trends, and category performance.

Key Features:

KPI Metrics (Sales, Profit, Quantity, AOV)
State-wise Sales Analysis
Monthly Profit Trends
Payment Mode Distribution
Customer-wise Sales Insights
Category & Sub-category Performance
Interactive Filters and Slicers

Tools Used:
Power BI | Data Visualization | DAX | Data Cleaning | Business Intelligence
